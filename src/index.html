Animated Hamburger Menu Button

I did the math to figure out the animations so you don't have to...

Examples:

Animated left-arrow:

    <button class="hamburger-button hamburger-animated-la">
    <span></span>
    </button>

Animated right-arrow:

    <button class="hamburger-button hamburger-animated-ra">
    <span></span>
    </button>

Animated "X":

    <button class="hamburger-button hamburger-animated-x">
    <span></span>
    </button>

Not Animated, just a hamburger:

    <button class="hamburger-button">
    <span></span>
    </button>

Installation

Using Bower:

$bower install hamburger-menu-button --save

In your html header:

<link href="bower_components/dist/hmb.css" rel="stylesheet">

In the footer:

<script src="bower_components/dist/hmb.js"></script>

Manual Installation:

Download the latest release from git: hmb-0.1.0.zip

Unzip the file and copy hmb.css and hmb.js to your working directory, then include them in your html:

In your html header:

<link href="path/to/hmb.css" rel="stylesheet">

In the footer:

<script src="path/to/hmb.js"></script>

Customization

The button is completely customizable via SCSS. Instead of directly including the hmb.css file in your code, you will need to include the source in your SCSS file.
